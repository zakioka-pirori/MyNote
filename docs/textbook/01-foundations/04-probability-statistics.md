# 第 4 章 確率と統計

## 学習目標

- 確率を「不確実性を数値化する言語」として体系的に理解する。
- 期待値・分散・主要な分布・大数の法則と中心極限定理を使えるようにする。
- 推定・仮説検定・ベイズ推論といった統計的推論の枠組みを理解する。
- 機械学習・A/B テスト・性能評価・確率的アルゴリズムの基盤を獲得する。

## 4.1 なぜ確率と統計か

CS の現場には不確実性が満ちている: A/B テスト、機械学習の予測、ネットワーク遅延の分布、ハッシュ衝突、暗号の鍵生成、ランダム化アルゴリズム、強化学習。**「正しいか間違っているか」ではなく「どれくらい確かか」を測る道具**が確率統計である。

## 4.2 確率の基礎

**標本空間** $\Omega$ は試行の結果の集合。**事象** は $\Omega$ の部分集合。確率測度 $P$ は次を満たす関数:

1. $0 \leq P(A) \leq 1$
2. $P(\Omega) = 1$
3. 互いに素な $A_1, A_2, \ldots$ について $P(\bigcup A_i) = \sum P(A_i)$（可算加法性）

これを **コルモゴロフの公理** という。

### 条件付き確率とベイズの定理

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

**ベイズの定理**:
$$P(A \mid B) = \frac{P(B \mid A) P(A)}{P(B)}$$

スパムフィルタ、医療診断、機械学習の確率モデルの根幹。

例（医療診断）: 罹患率 1%、検査の感度 99%・特異度 99% のとき、陽性結果が出ても本当に病気である確率は約 50% に過ぎない（基準率の誤謬）。

### 独立性

$P(A \cap B) = P(A) P(B)$ のとき $A, B$ は独立。条件付き独立 $P(A \cap B \mid C) = P(A \mid C) P(B \mid C)$ はベイジアンネットの本質。

## 4.3 確率変数

**確率変数** $X$ は標本空間から実数への関数。離散型と連続型がある。

- 離散: 確率質量関数 $p(x) = P(X = x)$
- 連続: 確率密度関数 $f(x)$、$P(a \leq X \leq b) = \int_a^b f(x)\,dx$

**累積分布関数** $F(x) = P(X \leq x)$ はどちらも定義可能。

### 期待値・分散

$$E[X] = \sum x p(x) \quad \text{または} \quad \int x f(x)\,dx$$
$$\operatorname{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2$$

期待値の線形性 $E[aX + bY] = aE[X] + bE[Y]$ は、$X, Y$ の独立性に依らず成立する強力な道具。

**共分散** $\operatorname{Cov}(X, Y) = E[(X - \mu_X)(Y - \mu_Y)]$、**相関係数** $\rho = \operatorname{Cov}(X,Y) / (\sigma_X \sigma_Y)$。$|\rho| \leq 1$。

## 4.4 主要な分布

### 離散分布

- **ベルヌーイ** $\mathrm{Bern}(p)$: 0/1 の試行。$E = p$, $\operatorname{Var} = p(1-p)$。
- **二項** $\mathrm{Bin}(n, p)$: 独立 $n$ 回中の成功数。$E = np$, $\operatorname{Var} = np(1-p)$。
- **幾何** $\mathrm{Geo}(p)$: 初めての成功までの試行数。
- **ポアソン** $\mathrm{Po}(\lambda)$: 単位時間あたりの稀な事象数。$E = \operatorname{Var} = \lambda$。アクセス数モデル、欠陥率。

### 連続分布

- **一様** $U(a, b)$: $f = 1/(b-a)$。
- **指数** $\mathrm{Exp}(\lambda)$: 次のイベントまでの時間。無記憶性 $P(T > s+t \mid T > s) = P(T > t)$。
- **正規** $N(\mu, \sigma^2)$: $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-(x-\mu)^2/(2\sigma^2)}$。標準化 $Z = (X - \mu)/\sigma \sim N(0,1)$。
- **ガンマ・ベータ・カイ二乗・$t$ 分布・$F$ 分布**: 統計推論で頻出。

## 4.5 大数の法則と中心極限定理

### 大数の法則 (LLN)

独立同分布の $X_1, \ldots, X_n$（期待値 $\mu$）について、標本平均 $\bar X_n \to \mu$（$n \to \infty$）。「データを集めれば真の値に近づく」が定理として保証される。

### 中心極限定理 (CLT)

$\bar X_n$ の分布は $n$ が大きいとき近似的に $N(\mu, \sigma^2/n)$。**元の分布によらず正規分布が現れる**ことが驚異的で、信頼区間・仮説検定・ガウシアンノイズ仮定の根拠となる。

## 4.6 推定

母集団パラメータ $\theta$ をデータから推定する。

### 最尤推定 (MLE)

尤度 $L(\theta) = \prod p(x_i; \theta)$ を最大化する $\hat\theta$。対数尤度を最大化することが多い。線形回帰、ロジスティック回帰、ニューラルネットの学習はすべて MLE の特殊形と見なせる。

### ベイズ推定

事後分布 $p(\theta \mid \mathcal{D}) \propto p(\mathcal{D} \mid \theta) p(\theta)$ を計算。事前分布 $p(\theta)$ を選ぶ点が哲学的に MLE と異なる。

### 不偏性・一致性・効率性

- 不偏: $E[\hat\theta] = \theta$。
- 一致: $\hat\theta \to \theta$（確率収束）。
- 標本分散 $\frac{1}{n-1} \sum (x_i - \bar x)^2$ で $n-1$ で割るのは不偏推定にするため。

## 4.7 信頼区間

「真値が含まれる区間を 95% の確率で当てる」推定。標本平均の場合 $\bar X \pm 1.96 \cdot s/\sqrt{n}$（$n$ が大きいとき）。

## 4.8 仮説検定

### 流れ

1. 帰無仮説 $H_0$ と対立仮説 $H_1$ を設定。
2. 検定統計量を計算。
3. $H_0$ のもとでの $p$ 値（観測値以上に極端な値が出る確率）を計算。
4. $p < \alpha$（典型 0.05）なら $H_0$ を棄却。

### 第 1 種・第 2 種誤り

- 第 1 種: $H_0$ が真なのに棄却（偽陽性）。
- 第 2 種: $H_1$ が真なのに棄却できず（偽陰性）。

A/B テストでは第 1 種は「効果なしを効果ありと判定」、第 2 種は「効果ありを見逃す」。

### よく使う検定

- $z$ 検定・$t$ 検定（平均の比較）
- カイ二乗検定（独立性、適合度）
- ANOVA（分散分析）
- マン-ホイットニーの U 検定（ノンパラメトリック）

## 4.9 多変量と回帰

**線形回帰** $y = \mathbf{w}^T \mathbf{x} + \varepsilon$。最小二乗解 $\hat{\mathbf{w}} = (X^T X)^{-1} X^T \mathbf{y}$。これは MLE（誤差が正規分布）と一致する。

**多重共線性** に注意。

**ロジスティック回帰** $P(y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^T \mathbf{x})$（$\sigma$ はシグモイド）。

## 4.10 マルコフ連鎖

「次の状態が現在の状態のみで決まる」確率過程。遷移行列 $P$ で記述。定常分布 $\pi P = \pi$。

応用: PageRank、隠れマルコフモデル (HMM)、強化学習の MDP、MCMC（モンテカルロ法）。

## 4.11 情報理論との接続

- **エントロピー** $H(X) = -\sum p(x) \log p(x)$: 不確かさの量。
- **KL ダイバージェンス** $D_{\mathrm{KL}}(p \| q) = \sum p \log(p/q)$: 分布間の距離（非対称）。
- **クロスエントロピー** $H(p, q) = H(p) + D_{\mathrm{KL}}(p \| q)$: 分類問題の損失関数。

これらは機械学習・データ圧縮・暗号の安全性指標で必須。

## 4.12 統計的機械学習との橋渡し

- バイアス・バリアンス分解: 期待二乗誤差 = バイアス² + バリアンス + ノイズ。
- 過学習・正則化（L1/L2）はベイズの事前分布として解釈できる。
- 交差検証は汎化誤差の推定。

## 4.13 演習

1. コインを 10 回投げて表が 7 回以上出る確率を求めよ。
2. ある検査の感度 95%、特異度 90%、罹患率 1% のとき、陽性者が病気である確率（陽性的中率）をベイズの定理で求めよ。
3. $X \sim N(0, 1)$ について $P(-1 \leq X \leq 1)$ を概算せよ。
4. $X_1, \ldots, X_n$ が $\mathrm{Bern}(p)$ のとき MLE が $\bar X$ になることを示せ。
5. A/B テストで A 群 1000 人中 50 人、B 群 1000 人中 70 人がコンバージョンしたとき、有意差があるか $z$ 検定で判定せよ。
6. 2 状態のマルコフ連鎖 $P = \begin{pmatrix}0.7 & 0.3 \\ 0.4 & 0.6\end{pmatrix}$ の定常分布を求めよ。

## 4.14 まとめ

確率は不確実性の代数、統計はデータから世界を推論する科学。期待値・分散・正規分布・大数の法則・中心極限定理・最尤推定・ベイズの定理・仮説検定が「7 つ道具」。これらは A/B テスト、機械学習、ネットワーク性能評価、ランダム化アルゴリズム、暗号の安全性議論など、CS 全域に広がる。直感だけで判断せず、数値的に「どれくらい確かか」を語れる態度を身につけたい。
