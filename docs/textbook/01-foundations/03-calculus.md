# 第 3 章 微積分

## 学習目標

- 「変化の割合」と「累積の量」という双対的な視点で微分・積分を理解する。
- 多変数の微積分を学び、勾配・ヘッセ行列・テイラー展開といった機械学習の必須道具を獲得する。
- 凸関数と勾配降下法を通じて、最適化の理論的枠組みを身につける。

## 3.1 なぜ微積分か

CS の中で連続数学が必要になる場面: 機械学習（損失関数の最小化）、コンピュータグラフィックス（曲線・曲面）、シミュレーション、信号処理、最適化、暗号の楕円曲線、確率論の連続分布。**深層学習の誤差逆伝播は、本質的には連鎖律という 1 行の数学**。

## 3.2 極限と連続

数列 $\{a_n\}$ が $L$ に **収束** するとは、任意の $\varepsilon > 0$ に対し、ある $N$ が存在して $n \geq N$ で $|a_n - L| < \varepsilon$ となること。

関数の極限 $\lim_{x \to a} f(x) = L$ も同様に $\varepsilon$-$\delta$ で定義される。

連続: $\lim_{x \to a} f(x) = f(a)$。

## 3.3 微分

導関数の定義:

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

幾何的には $x$ における接線の傾き、運動学的には瞬間変化率。

### 微分公式

- $(x^n)' = nx^{n-1}$
- $(\sin x)' = \cos x$, $(\cos x)' = -\sin x$
- $(e^x)' = e^x$, $(\ln x)' = 1/x$
- 和の微分: $(f+g)' = f' + g'$
- 積の法則: $(fg)' = f'g + fg'$
- 商の法則: $(f/g)' = (f'g - fg')/g^2$
- **連鎖律 (chain rule)**: $(f(g(x)))' = f'(g(x)) g'(x)$

連鎖律は**深層学習の誤差逆伝播の数学的本体**。合成関数の微分が「外側 × 内側」で進められるからこそ、ニューラルネットの勾配計算がバケツリレー的に実行できる。

### 平均値の定理

$f$ が $[a, b]$ で連続、$(a, b)$ で微分可能なら、$\exists c \in (a,b)$ で $f'(c) = \frac{f(b) - f(a)}{b - a}$。

ロルの定理（特殊形）と並んで、関数の振る舞いを微分から推論する基礎。

## 3.4 積分

**定積分** $\int_a^b f(x)\,dx$ は曲線下の面積（リーマン和の極限）。

**微積分学の基本定理**:
$$\frac{d}{dx} \int_a^x f(t)\,dt = f(x), \quad \int_a^b f'(x)\,dx = f(b) - f(a)$$

「微分と積分は逆操作」という主張で、これによって積分の計算が原始関数探しに帰着する。

### 主要技法

- 置換積分（変数変換）
- 部分積分: $\int u\,dv = uv - \int v\,du$
- 部分分数分解（有理関数）

## 3.5 級数と収束

無限和 $\sum_{n=1}^\infty a_n$ が収束するか発散するか。判定法:

- 比較判定・比判定 (d'Alembert)・根判定 (Cauchy)
- 交代級数（ライプニッツ）
- 絶対収束 ⇒ 収束

**べき級数** $\sum a_n (x-c)^n$ は収束半径を持ち、内部で項別微分・積分が許される。

### テイラー展開

滑らかな関数を多項式で近似:

$$f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots$$

例: $e^x = \sum \frac{x^n}{n!}$、$\sin x, \cos x$、$\ln(1+x)$。

機械学習の最適化で「ある点まわりで関数を 2 次式で近似する」発想（ニュートン法・ヘッセ行列）の根拠がこれ。

## 3.6 多変数微分

関数 $f(x, y)$ の **偏微分** は他の変数を定数とみなして微分:

$$\frac{\partial f}{\partial x}, \quad \frac{\partial f}{\partial y}$$

**勾配 (gradient)** はすべての偏微分を並べたベクトル:

$$\nabla f = \left(\frac{\partial f}{\partial x_1}, \ldots, \frac{\partial f}{\partial x_n}\right)$$

幾何的には「$f$ がもっとも増加する方向」。機械学習の「**勾配降下法**」では $\nabla f$ の逆方向に進んで関数を最小化する。

### ヘッセ行列

2 階偏微分を並べた対称行列:

$$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$$

固有値がすべて正なら極小、すべて負なら極大、混在なら鞍点。最適化と物理シミュレーションに登場。

### 多変数のテイラー展開

$$f(\mathbf{x}+\mathbf{h}) \approx f(\mathbf{x}) + \nabla f^T \mathbf{h} + \tfrac{1}{2} \mathbf{h}^T H \mathbf{h}$$

ニュートン法はこの 2 次近似を解いて更新する。

## 3.7 重積分とヤコビアン

多変数の積分は領域上の和:

$$\iint_D f(x,y)\,dx\,dy$$

変数変換 $\mathbf{x} = \Phi(\mathbf{u})$ では **ヤコビ行列の行列式** が体積要素のスケーリングを担う:

$$dx\,dy = |\det J_\Phi|\, du\,dv$$

確率密度の変換、グラフィックスの座標変換で必須。

## 3.8 微分方程式入門

$dy/dx = f(x, y)$ の形。物理シミュレーション、強化学習の連続時間版、神経モデル、暗号の楕円曲線群法則の背景にある。

代表的な解法:
- 変数分離: $y' = g(x)h(y)$
- 線形 1 階: 積分因子法
- 数値解法: オイラー法、ルンゲ-クッタ法

## 3.9 凸最適化の入門

**凸関数**: 任意の $0 \leq t \leq 1$ で $f(t\mathbf{x} + (1-t)\mathbf{y}) \leq tf(\mathbf{x}) + (1-t)f(\mathbf{y})$。

凸関数は「お椀型」で、極小 = 大域最小という良い性質がある。線形回帰の二乗誤差・ロジスティック回帰の対数尤度・SVM のヒンジ損失はすべて凸。

### 勾配降下法 (Gradient Descent)

$$\mathbf{x}_{k+1} = \mathbf{x}_k - \eta \nabla f(\mathbf{x}_k)$$

$\eta$ は学習率。凸関数で $\eta$ が適切なら大域最適に収束する。バリエーション:
- 確率的勾配降下法 (SGD)
- モメンタム法
- Adam

### ニュートン法

$$\mathbf{x}_{k+1} = \mathbf{x}_k - H^{-1} \nabla f$$

2 階情報を使うので収束が速いが計算コストが高い。

### ラグランジュ未定乗数法

制約 $g(\mathbf{x}) = 0$ のもとで $f$ を最適化するとき、$\nabla f = \lambda \nabla g$ を解く。SVM や経済学の最適化で必須。

## 3.10 数値計算上の注意

- 浮動小数点表現と桁落ち（$1 + 10^{-16} = 1$ になる）。
- 対数損失のソフトマックスでは「log-sum-exp トリック」で安定化する。
- 大きな数の指数は `expm1`、`log1p` で。
- 自動微分（PyTorch / JAX）は連鎖律をプログラム的に展開する仕組み。

## 3.11 演習

1. $f(x) = x^3 - 3x + 1$ の極値を求めよ。
2. $\int_0^1 x e^x\,dx$ を部分積分で計算せよ。
3. $\sin x$ のテイラー展開を $x = 0$ まわりで 5 次まで求めよ。
4. $f(x, y) = x^2 + xy + y^2$ の勾配・ヘッセ行列を求め、$(0, 0)$ での極値判定をせよ。
5. 平均値の定理を用いて、$|\sin a - \sin b| \leq |a - b|$ を示せ。
6. 学習率 $\eta = 0.1$ で $f(x) = x^2$ に勾配降下法を適用し、$x_0 = 5$ から 5 ステップの軌跡を計算せよ。

## 3.12 まとめ

微積分は「変化を捉える数学」。1 変数では微分が傾き・積分が面積で、両者は基本定理で結ばれる。多変数では勾配・ヘッセ・ヤコビアンが主役で、機械学習の損失最小化や物理シミュレーションを駆動する。連鎖律と凸最適化は CS 応用での最重要トピックなので、計算技法だけでなく図形的直感をしっかり押さえること。
