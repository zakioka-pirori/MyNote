# 第 17 章 人工知能と機械学習

## 学習目標

- 古典的 AI（探索・推論）から統計的機械学習・深層学習・LLM までを 1 本の流れとして理解する。
- 教師あり/教師なし/強化学習の枠組みと、代表的アルゴリズムを使い分けられる。
- 過学習・正則化・評価指標・モデル選択を実務的に判断できる。
- ニューラルネットの誤差逆伝播、CNN, RNN, Transformer の基本構造を説明できる。

## 17.1 なぜ AI/ML を学ぶか

AI はもはや特殊技術ではなく、ほぼすべてのアプリに何らかの形で組み込まれる **基本要素技術** になった。検索, 推薦, 翻訳, 異常検知, 自動運転, 音声・画像認識, 生成 AI。エンジニアが API 利用者で終わらず、自分でモデルを訓練・評価・デプロイできるようになると、解ける問題の範囲が一気に広がる。

## 17.2 AI の歴史的全体像

- 1950s 記号 AI: チューリングテスト、論理推論。
- 1960s パーセプトロン、ELIZA。
- 1980s エキスパートシステム、第二次 AI ブーム。
- 1990s 統計的機械学習、SVM。
- 2010s 深層学習（ImageNet 2012 が転機）。
- 2017 Transformer 登場 (Attention is All You Need)。
- 2020s LLM の時代 (GPT, Claude)。

## 17.3 探索

問題を「状態空間」のグラフ探索として定式化。

- 非情報的: BFS, DFS, 反復深化。
- 情報的（ヒューリスティック）: 貪欲ベスト・ファースト, A\*。
- 制約充足 (CSP): バックトラック + 推論（弧整合性 AC-3）。
- ゲーム探索: ミニマックス, αβ 枝刈り, モンテカルロ木探索 (MCTS)。AlphaGo の核。

## 17.4 機械学習の枠組み

### データから関数を学ぶ

- 教師あり: 入力 $x$ と正解 $y$ から $f$ を学ぶ。回帰・分類。
- 教師なし: $y$ なし、データの構造を発見。クラスタリング・次元削減。
- 強化学習: 環境との相互作用で報酬を最大化する方策を学ぶ。

### 学習の基本サイクル

1. データ収集と前処理。
2. 特徴量エンジニアリング（または学習）。
3. モデル選択。
4. 学習（損失最小化）。
5. 評価（汎化性能を別データで）。
6. デプロイと監視。

### 損失関数

- 二乗損失 (回帰): $(\hat y - y)^2$。
- クロスエントロピー (分類): $-\sum y \log \hat y$。
- ヒンジ損失 (SVM)。

### 過学習と正則化

- 訓練データに過剰適応すると未知データで性能が落ちる。
- 対策: より多いデータ、シンプルなモデル、L1/L2 正則化、ドロップアウト、早期終了、データ拡張。
- バイアス・バリアンス分解: 期待誤差 = バイアス² + バリアンス + ノイズ。

### 評価とデータ分割

- 訓練 / 検証 / テスト。
- 交差検証 (k-fold)。
- 漏洩 (leakage) を避ける。
- 評価指標:
  - 回帰: MSE, RMSE, MAE, R²。
  - 分類: Accuracy, Precision, Recall, F1, ROC-AUC, PR-AUC。
  - 不均衡データでは Accuracy が誤誘導 → F1 や AUC を見る。

## 17.5 古典機械学習

### 線形回帰

$\hat y = \mathbf{w}^T \mathbf{x} + b$。最小二乗解で閉じた形。仮定: 残差は正規分布で独立同分布。

### ロジスティック回帰

二値分類。$P(y=1|x) = \sigma(\mathbf{w}^T x)$。最大尤度推定で学習。線形分類器の代表。

### k-近傍 (kNN)

訓練データを覚えておき、近い k 個の多数決で予測。距離尺度がすべて。次元の呪いに注意。

### 決定木

軸並行な分割で空間を区切る。情報利得 / Gini 不純度で分岐。解釈性が高い。過学習しがち → 剪定。

### アンサンブル

- バギング: ランダムフォレスト。
- ブースティング: AdaBoost, Gradient Boosting, XGBoost, LightGBM, CatBoost。テーブルデータで最強クラス。
- スタッキング。

### サポートベクタマシン (SVM)

マージンを最大化する超平面。カーネル法で非線形（RBF, 多項式）。少データ高次元に強い。

### ナイーブベイズ

ベイズの定理 + 特徴独立仮定。テキスト分類で素朴だが強力。

### 教師なし

- クラスタリング: K-Means, DBSCAN, 階層クラスタリング。
- 次元削減: PCA, t-SNE, UMAP。
- 密度推定: ガウス混合モデル (GMM)、カーネル密度推定。

### 異常検知

Isolation Forest, One-Class SVM, Autoencoder。

## 17.6 ニューラルネットワーク

### パーセプトロン

$\hat y = \sigma(\mathbf{w}^T \mathbf{x} + b)$。線形分離可能な問題のみ解ける（XOR 解けない）。

### 多層パーセプトロン (MLP)

層を重ねて非線形を表現。万能近似定理: 1 隠れ層でも任意の連続関数を任意精度で近似できる。

### 活性化関数

- シグモイド: 飽和して勾配消失。
- tanh。
- ReLU: $\max(0, x)$。深層学習を実用化した立役者。
- Leaky ReLU, ELU, GELU, Swish。

### 誤差逆伝播 (Backpropagation)

連鎖律で勾配を出力層から入力層へ伝播。計算量は順伝播と同じ $O(n)$。

```
forward:  x → h₁ → h₂ → ŷ → L
backward: x ← h₁ ← h₂ ← ŷ ← L
          ∂L/∂x ← ∂L/∂h₁ ← …
```

自動微分（PyTorch の autograd, JAX の grad）でプログラム的に実装される。

### 最適化

- SGD（確率的勾配降下）。
- Momentum, Nesterov。
- Adam, AdamW（広く使われる）。
- 学習率スケジューリング（warmup, cosine）。

### 正則化

- ドロップアウト。
- バッチ正規化（BN）/ レイヤー正規化（LN）。
- 重み減衰。
- データ拡張。

### 勾配の問題

- 消失 / 爆発: 残差接続、適切な初期化（He, Xavier）、勾配クリッピング。

## 17.7 畳み込みニューラルネット (CNN)

画像など局所性のあるデータに有効。

- 畳み込み層: 小さなフィルタを移動。重み共有でパラメータ削減。
- プーリング: 最大値・平均で空間を縮約。
- アーキテクチャ: LeNet, AlexNet, VGG, ResNet (残差接続), EfficientNet。
- 応用: 物体検出 (Faster R-CNN, YOLO), セグメンテーション (U-Net)。

## 17.8 リカレント NN (RNN)

時系列・系列を扱う。

- バニラ RNN: 勾配消失で長期依存を学べない。
- LSTM: ゲート機構（忘却・入力・出力）で長期依存。
- GRU: LSTM の簡略版。
- 系列ラベリング, 翻訳, 音声認識。

## 17.9 Attention と Transformer

### Attention

「クエリと最も関連するキーから値を取り出す」加重和。長距離依存を 1 ステップで参照できる。

### Self-Attention

入力系列が自身を参照。$\text{Attention}(Q, K, V) = \text{softmax}(QK^T/\sqrt{d_k}) V$。

### Multi-Head Attention

複数の注意を並列に。多様な関係を捉える。

### Transformer

エンコーダ・デコーダ構成。位置エンコーディング, 残差接続, レイヤー正規化, FFN。BERT (エンコーダのみ), GPT (デコーダのみ), T5 (両方)。

### スケーリング則

パラメータ・データ・計算量を増やすと性能が冪乗で改善する（Kaplan et al.）。これが LLM 競争の根拠。

## 17.10 大規模言語モデル (LLM)

- 事前学習: 大規模コーパスで次トークン予測。
- ファインチューニング: タスク特化。
- 指示学習 (Instruction Tuning), RLHF (人間フィードバック強化学習)。
- プロンプトエンジニアリング, Few-Shot, Chain-of-Thought。
- RAG (Retrieval-Augmented Generation): 外部知識を検索して与える。
- ツール使用、Agent。
- 評価: ベンチマーク (MMLU, HumanEval), 人手評価。
- 安全性: 幻覚 (hallucination), プロンプトインジェクション、バイアス, アライメント。

## 17.11 強化学習

エージェントが状態から行動を選び、報酬を得て方策を改善。

- マルコフ決定過程 (MDP): 状態, 行動, 遷移確率, 報酬, 割引率。
- 価値反復, 方策反復。
- Q 学習, SARSA。
- 関数近似（ニューラルネット）→ DQN（Atari）。
- 方策勾配 (REINFORCE), Actor-Critic, PPO, SAC。
- 探索と活用のトレードオフ (ε-greedy, UCB)。
- AlphaGo / AlphaZero: MCTS + RL + 自己対戦。

## 17.12 生成モデル

- VAE: 潜在変数モデル + 変分推論。
- GAN: 生成器と識別器の対立。
- Flow ベース: 可逆な変換で密度を直接計算。
- 拡散モデル (Diffusion): ノイズ付加 → 逆向きに復元。Stable Diffusion, DALL-E 2, Imagen の核。

## 17.13 機械学習システム (MLOps)

学習だけでなく運用が本番。

- 特徴量ストア (Feast)。
- モデルレジストリ (MLflow)。
- パイプライン (Kubeflow, Airflow, Prefect)。
- 推論サービング (Triton, TorchServe, vLLM)。
- モデル監視: データドリフト, 概念ドリフト, 性能劣化。
- A/B テスト, シャドーデプロイ, カナリア。
- 再現性: コード/データ/環境の固定 (DVC, lockfile)。

## 17.14 倫理と責任ある AI

- 公平性 (Fairness): 群間で性能・誤りが偏らない。
- 説明可能性 (XAI): SHAP, LIME, グラフィカルモデル。
- プライバシ: 差分プライバシ, 連合学習。
- 透明性とアカウンタビリティ。
- 利用者への影響と社会的責任。

## 17.15 演習

1. アヤメデータでロジスティック回帰を実装し、Accuracy, Precision, Recall, F1 を計算せよ。
2. 線形回帰の正規方程式 $\mathbf{w} = (X^TX)^{-1}X^T\mathbf{y}$ を導出せよ。
3. 二値分類で陽性が 1%、再現率 95%, 適合率 50% のとき、F1 を求めよ。
4. ReLU と Sigmoid を比較し、深層 NN で ReLU が好まれる理由を述べよ。
5. CNN がパラメータ数を MLP より大幅に削減できる仕組みを説明せよ。
6. Self-Attention の計算量が系列長 $n$ に対して $O(n^2)$ である理由と、長文化の課題を述べよ。
7. 強化学習の Q 学習の更新式 $Q(s,a) \leftarrow Q(s,a) + \alpha [r + \gamma \max Q(s',a') - Q(s,a)]$ の各項の意味を述べよ。
8. データ漏洩 (data leakage) の例を 2 つ挙げ、防ぎ方を述べよ。

## 17.16 まとめ

機械学習は「データから規則を学ぶ」枠組みで、確率統計・線形代数・最適化・微積分のすべてが交差する。古典的 ML はテーブルデータで今も主役、深層学習は画像・音声・自然言語・生成で支配的、LLM はソフトウェアとの新たな接点を生んだ。**評価と監視が学習と同じくらい重要** であり、ベンチマークの数字を盲信せず、実環境での挙動と倫理に責任を持てるエンジニアになること。
