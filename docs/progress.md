# 学習の記録

教科書のどこまで読んだかをチェックして、**全体の進捗** を一目で確認できるページです。

## 使い方

- 各章のページ上部にある「**この章を読了にする**」にチェックを入れると、ここに反映されます。
- このページのチェックボックスからも直接 ON/OFF できます。
- 各章ページの「**📝 この章のメモ**」欄に、気づき・疑問・復習ポイントを書き残せます（メモがある章には 📝 が表示されます）。

## 保存と端末間の同期

進捗とメモの保存先は 2 通りあります。

| モード | 保存先 | 端末間の同期 | 設定 |
|---|---|---|---|
| **既定** | ブラウザの localStorage | なし | 不要 |
| **Gist 同期** | GitHub の非公開 Gist | あり（自動） | トークンを 1 度設定 |

### 端末間で同期したい場合

下のダッシュボード内「**🔄 端末間で同期する（GitHub Gist）**」を開き、画面の手順に従って GitHub の個人アクセストークン（スコープは `gist` のみ）を設定してください。

- 一度設定すれば、チェックやメモ編集の数秒後に **自動でアップロード**、ページを開くと **自動で取得** されます。
- **別の端末でも同じトークンを入れるだけ** で、同じ Gist が自動的に見つかり同期されます。
- データは **非公開 (secret) の Gist** に保存され、第三者には見えません。

!!! warning "トークンの取り扱い"
    - トークンはブラウザの localStorage に保存されます。**共有 PC では使わないでください。**
    - スコープは必ず `gist` のみにしてください（他の権限は不要です）。
    - 同期を設定しない場合、進捗・メモはそのブラウザの中だけに保存され、閲覧データを消すと失われます。
