# 高校数学 自習教科書

中学数学の基礎しか覚えていない大人・社会人・高校生のために、**ゼロから共通テスト 80 点以上**を目指せるよう書き下ろした 6 冊の教科書。

## この教科書を作った想い

「大人になって数学をもう一度学び直したい」「子供の宿題を見てやれるようになりたい」「データサイエンスや AI に必要な数学の基礎を整えたい」――そんな人のために作りました。

中学数学（一次方程式、ピタゴラスの定理、平方根、簡単な確率）レベルから読み始めても理解できるよう、**1 つひとつの式を「なぜそうなるか」から丁寧に解説** しています。

## 共通テスト 80 点以上のロードマップ

```
数学 I・A 試験 (60 + 60 点 = 100 点 → 80 点を目標)
   ↑
数学 I + 数学 A を完走

数学 II・B・C 試験 (100 点満点 → 80 点を目標)
   ↑
数学 II + 数学 B + 数学 C を完走

(理系志望なら) 数学 III も
```

## 学習の進め方

1. **順番に読む**。数学 I → A → II → B → III → C の順が王道。
2. **必ず手を動かす**。計算過程は紙とペンで再現する。読むだけは厳禁。
3. **例題を「解いてから」答えを見る**。
4. **章末問題は全問解く**。間違えたら本文に戻る。
5. **共通テスト過去問** を併用。各単元終了ごとに該当問題を解く。

## 中学数学の前提

以下が「不安なく扱える」ことを前提にしています。

- 文字式の計算（展開、因数分解の基礎）
- 一次方程式・連立方程式
- 比例・反比例
- 平方根、ピタゴラスの定理
- 三角形の合同・相似
- 円の性質（中心角・円周角）
- 簡単な確率（コイン、サイコロ）
- ヒストグラム、平均値

不安があれば、本書中に **「中学数学の復習」** ボックスを挟んでいるので、その都度確認できます。
