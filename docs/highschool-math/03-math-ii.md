# 数学 II

## まえがき

数学 II は **数学 I を一気に拡張する** 巻。

- 式の世界: 多項式の割り算、複素数
- 図形の世界: 座標平面、円の方程式、軌跡
- 関数の世界: 三角・指数・対数の世界へ
- 解析の世界: 微分・積分の入門

「**範囲が広い**」のが特徴ですが、各分野は独立しているので順番に攻略可能。共通テスト 数学 II・B・C で **配点の大半** を占めます。

## 第 1 章 式と証明

### 1.1 多項式の割り算

整数の割り算と同じ手順:
$$A = B \cdot Q + R$$
（$A$ を $B$ で割った商 $Q$、余り $R$）

例: $x^3 - 2x^2 + 3$ を $x - 1$ で割る
- $x^3 \div x = x^2$ → $x^2(x-1) = x^3 - x^2$ を引く
- $-x^2 + \cdots$ を $x$ で割って...
- 結果: 商 $x^2 - x - 1$, 余り $2$

#### 1.1.1 剰余の定理

「**$P(x)$ を $x - a$ で割った余り = $P(a)$**」

例: $P(x) = x^3 - 2x^2 + 3$ を $x - 1$ で割った余り = $P(1) = 1 - 2 + 3 = 2$ ✓

#### 1.1.2 因数定理

$P(a) = 0$ ⇔ $P(x)$ は $x - a$ で割り切れる

これを使って **高次方程式を因数分解** できます。

例: $x^3 - 6x^2 + 11x - 6 = 0$
- $P(1) = 0$ → $(x - 1)$ で割れる
- $P(x) = (x - 1)(x^2 - 5x + 6) = (x-1)(x-2)(x-3)$
- 解: $x = 1, 2, 3$

### 1.2 分数式

分母に文字を含む式の計算。「**通分してまとめる**」が基本。

### 1.3 恒等式と等式の証明

#### 1.3.1 恒等式

「**任意の値で成り立つ式**」。係数比較で値を決める。

例: $a x^2 + b x + c = (x - 1)(x - 2) + d$ が恒等式 → 展開して係数を比較。

#### 1.3.2 等式の証明

- 左辺 = 右辺 を直接示す
- 両辺を計算して同じ値に
- $A = B$ ⇔ $A - B = 0$ を示す

### 1.4 不等式の証明

#### 1.4.1 相加平均と相乗平均

$a, b > 0$ について:
$$\frac{a + b}{2} \geq \sqrt{ab}$$
等号は $a = b$ のとき。

応用例: $x > 0$ で $x + \frac{1}{x}$ の最小値
$$x + \frac{1}{x} \geq 2\sqrt{x \cdot \frac{1}{x}} = 2$$
最小値は $x = 1$ のとき $2$。

#### 1.4.2 コーシー・シュワルツの不等式

$$(a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2$$

### 1.5 複素数

$i^2 = -1$ を満たす数を導入。

#### 1.5.1 基本

$$a + bi \quad (a, b \in \mathbb{R})$$

- 実部 $a$, 虚部 $b$
- 共役 $\overline{a + bi} = a - bi$
- 絶対値 $|a + bi| = \sqrt{a^2 + b^2}$

#### 1.5.2 計算

$$(a + bi) + (c + di) = (a+c) + (b+d)i$$
$$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$$

割り算は共役を分母分子に掛けて有理化:
$$\frac{1}{2 + i} = \frac{2 - i}{(2+i)(2-i)} = \frac{2 - i}{5}$$

#### 1.5.3 2 次方程式の複素解

判別式 $D < 0$ なら複素解:
$$x = \frac{-b \pm i\sqrt{4ac - b^2}}{2a}$$

## 第 2 章 図形と方程式

### 2.1 点と直線

#### 2.1.1 内分・外分

2 点 $A(x_1, y_1)$, $B(x_2, y_2)$ を $m:n$ に内分する点:
$$\left( \frac{n x_1 + m x_2}{m + n}, \frac{n y_1 + m y_2}{m + n} \right)$$

#### 2.1.2 直線の方程式

- 傾き $m$, $y$ 切片 $n$: $y = mx + n$
- 2 点 $(x_1, y_1), (x_2, y_2)$ を通る: $y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)$
- 一般形: $ax + by + c = 0$

#### 2.1.3 平行・垂直

- 平行: 傾きが等しい
- 垂直: 傾きの積が $-1$

#### 2.1.4 点と直線の距離

点 $(x_0, y_0)$ から直線 $ax + by + c = 0$ までの距離:
$$d = \frac{|a x_0 + b y_0 + c|}{\sqrt{a^2 + b^2}}$$

### 2.2 円

#### 2.2.1 円の方程式

中心 $(a, b)$, 半径 $r$:
$$(x - a)^2 + (y - b)^2 = r^2$$

展開形:
$$x^2 + y^2 + lx + my + n = 0$$

#### 2.2.2 円と直線の位置関係

中心から直線までの距離 $d$ と半径 $r$ で:
- $d < r$: 交わる（2 点）
- $d = r$: 接する（1 点）
- $d > r$: 交わらない

#### 2.2.3 接線の方程式

円 $x^2 + y^2 = r^2$ 上の点 $(x_0, y_0)$ における接線:
$$x_0 x + y_0 y = r^2$$

### 2.3 軌跡と領域

#### 2.3.1 軌跡

「**ある条件を満たす点の集合**」。

例: $A(-2, 0), B(2, 0)$ からの距離の比が 2:1 の点の軌跡
- $\sqrt{(x+2)^2 + y^2} : \sqrt{(x-2)^2 + y^2} = 2:1$
- 整理すると円

#### 2.3.2 領域

不等式が表す **領域** を図示:
- $y > x + 1$: 直線の上側
- $x^2 + y^2 < 4$: 円の内部

#### 2.3.3 線形計画法

「**目的関数を最大化（最小化）する**」問題。共通テスト頻出。

例: $x + 2y \leq 8$, $3x + y \leq 9$, $x, y \geq 0$ のもとで、$z = 4x + 5y$ の最大値
- 領域を図示
- $z$ の等値線を平行移動
- 領域の頂点で最大に

## 第 3 章 三角関数

数学 I の三角比を **任意角・周期関数** へ拡張。

### 3.1 一般角と弧度法

#### 3.1.1 一般角

「**何回転もする角度**」も扱う。
- 反時計回りが正、時計回りが負
- $90° = 90° + 360°k$ などすべて同じ位置

#### 3.1.2 弧度法（ラジアン）

「**弧の長さで角を測る**」方法。

$$180° = \pi \text{ rad}$$

| 度 | ラジアン |
|---|---|
| 30° | $\pi/6$ |
| 45° | $\pi/4$ |
| 60° | $\pi/3$ |
| 90° | $\pi/2$ |
| 180° | $\pi$ |
| 360° | $2\pi$ |

#### 3.1.3 弧と面積

半径 $r$, 中心角 $\theta$（ラジアン）の扇形:
- 弧の長さ: $r\theta$
- 面積: $\frac{1}{2} r^2 \theta$

### 3.2 三角関数

単位円上の点 $(\cos\theta, \sin\theta)$ で定義（数学 I の拡張）。

#### 3.2.1 グラフ

```
sin x:      1 ┤     ／＼
              │   ／    ＼
            0 ┤ ─／──────＼──→ x
              │           ＼
           -1 ┤             ＼  ／
                              ＼／
              0  π/2  π  3π/2  2π
```

$\sin, \cos$ は周期 $2\pi$、$\tan$ は周期 $\pi$。

#### 3.2.2 重要な公式

- $\sin^2\theta + \cos^2\theta = 1$
- $\sin(-\theta) = -\sin\theta$, $\cos(-\theta) = \cos\theta$
- $\sin(\pi - \theta) = \sin\theta$, $\cos(\pi - \theta) = -\cos\theta$
- $\sin(\theta + \pi/2) = \cos\theta$, $\cos(\theta + \pi/2) = -\sin\theta$

### 3.3 加法定理 — 最重要公式

$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$
$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$
$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$
$$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$
$$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$

**覚え方**: 「**サイン: シェアハーフ + ハーフシェア**」「**コサイン: ココマイサシ**」など。

### 3.4 2 倍角・半角・3 倍角

加法定理から導出。

$$\sin 2\theta = 2\sin\theta\cos\theta$$
$$\cos 2\theta = \cos^2\theta - \sin^2\theta = 1 - 2\sin^2\theta = 2\cos^2\theta - 1$$

**半角公式**:
$$\sin^2\frac{\theta}{2} = \frac{1 - \cos\theta}{2}, \quad \cos^2\frac{\theta}{2} = \frac{1 + \cos\theta}{2}$$

### 3.5 三角関数の合成

$$a\sin\theta + b\cos\theta = \sqrt{a^2 + b^2} \sin(\theta + \alpha)$$
ここで $\tan\alpha = b/a$。

例: $\sin\theta + \sqrt 3 \cos\theta = 2\sin(\theta + \pi/3)$

最大値は $\sqrt{a^2 + b^2}$、最小値は $-\sqrt{a^2 + b^2}$。

### 3.6 三角方程式・不等式

例: $\sin\theta = 1/2$ ($0 \leq \theta < 2\pi$)
$\theta = \pi/6, 5\pi/6$

**単位円** で考えると、解が見つけやすい。

## 第 4 章 指数関数と対数関数

### 4.1 指数の拡張

中学では $a^n$ は「$a$ を $n$ 個掛ける」だけ。これを **負の数・分数・無理数** へ拡張。

#### 4.1.1 0 乗・負の指数

$$a^0 = 1, \quad a^{-n} = \frac{1}{a^n}$$

#### 4.1.2 有理数の指数

$$a^{1/n} = \sqrt[n]{a}, \quad a^{m/n} = \sqrt[n]{a^m}$$

例: $8^{2/3} = (\sqrt[3]{8})^2 = 4$

#### 4.1.3 指数法則

| 法則 | 式 |
|---|---|
| 積 | $a^m \cdot a^n = a^{m+n}$ |
| 商 | $a^m / a^n = a^{m-n}$ |
| 累乗 | $(a^m)^n = a^{mn}$ |
| 別底 | $(ab)^n = a^n b^n$ |

### 4.2 指数関数

$y = a^x$（$a > 0, a \neq 1$）。

```
a > 1:          0 < a < 1:
  y │  ／         y │＼
    │ ／            │ ＼
  1─┤／           1─┤  ＼
    │              │    ＼
  ──┼──→ x       ──┼──────→ x
    0              0
```

- $a > 1$: 増加関数
- $0 < a < 1$: 減少関数
- どちらも $(0, 1)$ を通る、$y > 0$

#### 4.2.1 指数方程式・不等式

底をそろえて指数比較:
$$a^p = a^q \Leftrightarrow p = q$$
$$a^p > a^q \Leftrightarrow p > q \text{ (a > 1 のとき)}$$

例: $2^{x+1} = 8$ → $2^{x+1} = 2^3$ → $x = 2$

### 4.3 対数

「**指数の逆**」。

$$y = a^x \Leftrightarrow x = \log_a y$$

「$a$ を **何乗** すると $y$ になるか」が $\log_a y$。

#### 4.3.1 基本

- $\log_a 1 = 0$
- $\log_a a = 1$
- $a^{\log_a x} = x$
- $\log_a a^x = x$

#### 4.3.2 対数法則

$$\log_a (MN) = \log_a M + \log_a N$$
$$\log_a (M/N) = \log_a M - \log_a N$$
$$\log_a M^n = n \log_a M$$
$$\log_a M = \frac{\log_b M}{\log_b a} \quad (\text{底の変換})$$

### 4.4 対数関数

$y = \log_a x$（$a > 0, a \neq 1$, $x > 0$）。

指数関数の **逆関数**。$y = a^x$ のグラフを $y = x$ で対称に。

### 4.5 常用対数

底 10 の対数 $\log_{10}$。よく使う近似値:
$$\log_{10} 2 \approx 0.3010, \quad \log_{10} 3 \approx 0.4771$$

これがあれば、$\log_{10} 5 = \log_{10}(10/2) = 1 - 0.3010 = 0.6990$ など計算できます。

#### 4.5.1 桁数の問題

$N$ が $k$ 桁 ⇔ $10^{k-1} \leq N < 10^k$ ⇔ $k - 1 \leq \log_{10} N < k$

例: $2^{50}$ の桁数
$\log_{10} 2^{50} = 50 \cdot 0.3010 = 15.05$
$\lfloor 15.05 \rfloor + 1 = 16$ 桁

## 第 5 章 微分法

「**変化の割合**」を捉える数学。

### 5.1 平均変化率と微分係数

#### 5.1.1 平均変化率

$f(x)$ の $x = a$ から $x = b$ への平均変化率:
$$\frac{f(b) - f(a)}{b - a}$$

「**$x$ が $a$ から $b$ に変わるとき、$y$ がどれくらい変わったか**」。

#### 5.1.2 微分係数

平均変化率の **極限**:
$$f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

これは「**$x = a$ における接線の傾き**」。

### 5.2 導関数

各点 $x$ の微分係数を関数として考えたもの:
$$f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h}$$

#### 5.2.1 基本公式

$$(x^n)' = n x^{n-1}$$

| 関数 | 導関数 |
|---|---|
| $c$ (定数) | $0$ |
| $x$ | $1$ |
| $x^2$ | $2x$ |
| $x^3$ | $3x^2$ |
| $x^n$ | $n x^{n-1}$ |

#### 5.2.2 公式

- $(f + g)' = f' + g'$
- $(cf)' = c f'$
- $(fg)' = f'g + fg'$（積の法則）
- $(f/g)' = (f'g - fg')/g^2$（商の法則）

数学 II の範囲は **多項式の微分** が中心。

### 5.3 関数の増減と極値

#### 5.3.1 増減表

$f'(x)$ の符号で関数の増減が分かる:
- $f'(x) > 0$: 増加
- $f'(x) < 0$: 減少
- $f'(x) = 0$: 極値の候補

例: $f(x) = x^3 - 3x$
- $f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$
- $f' = 0$ で $x = -1, 1$

| $x$ | $\cdots$ | $-1$ | $\cdots$ | $1$ | $\cdots$ |
|---|---|---|---|---|---|
| $f'$ | + | 0 | - | 0 | + |
| $f$ | ↗ | 極大 2 | ↘ | 極小 -2 | ↗ |

#### 5.3.2 グラフを描く

増減表 + $y$ 切片 + $x$ 切片で 3 次関数のグラフを描く。

### 5.4 最大・最小

定義域内で **増減を調べ、端点と極値を比較**。

### 5.5 方程式・不等式への応用

例: $x^3 - 3x + k = 0$ が異なる 3 つの実数解 ⇔ ?
- $f(x) = x^3 - 3x$ のグラフを描く
- 極大 2、極小 -2
- 直線 $y = -k$ が 3 か所で交わる条件: $-2 < -k < 2$ → $-2 < k < 2$

## 第 6 章 積分法

### 6.1 不定積分

「**微分の逆**」:
$$\int f(x) dx = F(x) + C \Leftrightarrow F'(x) = f(x)$$

$C$ は積分定数（任意の定数）。

#### 6.1.1 基本公式

$$\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

例: $\int (3x^2 - 2x + 1) dx = x^3 - x^2 + x + C$

### 6.2 定積分

「**両端の値を引く**」:
$$\int_a^b f(x) dx = F(b) - F(a)$$

性質:
- $\int_a^b cf(x) dx = c \int_a^b f(x) dx$
- $\int_a^b (f + g) dx = \int f + \int g$
- $\int_a^b = -\int_b^a$
- $\int_a^a = 0$

### 6.3 面積

「**曲線と $x$ 軸で囲まれた面積**」:
$$S = \int_a^b f(x) dx \quad (f \geq 0)$$

2 つの曲線の間:
$$S = \int_a^b (f(x) - g(x)) dx \quad (f \geq g)$$

例: $y = x^2$ と $y = 2x$ で囲まれた面積
- 交点: $x^2 = 2x$ → $x = 0, 2$
- $0 \leq x \leq 2$ で $2x \geq x^2$
- $S = \int_0^2 (2x - x^2) dx = [x^2 - x^3/3]_0^2 = 4 - 8/3 = 4/3$

### 6.4 公式

- $\int_a^b (x - a)(x - b) dx = -\frac{(b - a)^3}{6}$（**1/6 公式**、頻出）
- $\int_a^b (x - a)^2 dx = \frac{(b - a)^3}{3}$

「**1/6 公式**」は 2 次関数と直線の囲む面積で時短になります。
