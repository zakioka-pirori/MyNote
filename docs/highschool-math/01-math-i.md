## 第 1 章 数と式

### 1.1 多項式の展開

#### 1.1.1 文字式の復習

文字式とは「**文字（変数）を含む式**」。中学で習った $2x + 3$、$x^2 - 4x + 4$ のようなものです。

「**同類項**」をまとめるルールを思い出しましょう:
$$3x + 5x = 8x \qquad 2x^2 + 4x^2 = 6x^2$$

ただし $x$ と $x^2$ は **別の項** なので、$3x + 2x^2$ はこれ以上まとめられません。

#### 1.1.2 展開の基本公式

**展開** とは、$( \cdot )$ を外して 1 つの多項式にすること。

**基本 1: 分配法則**
$$a(b + c) = ab + ac$$

例: $3(x + 2) = 3x + 6$

**基本 2: 2 項 × 2 項**
$$(a + b)(c + d) = ac + ad + bc + bd$$

例:
$$(x + 2)(x + 3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$$

#### 1.1.3 覚えるべき展開公式

中学でも一部習いましたが、ここで完全に身につけます。

| 公式 | 形 |
|---|---|
| 平方公式（和） | $(a + b)^2 = a^2 + 2ab + b^2$ |
| 平方公式（差） | $(a - b)^2 = a^2 - 2ab + b^2$ |
| 和と差の積 | $(a + b)(a - b) = a^2 - b^2$ |
| 3 項の平方 | $(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$ |
| 3 乗の和・差 | $(a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$ |
| 立方の和・差 | $(a + b)(a^2 - ab + b^2) = a^3 + b^3$ |

> **🌱 確認問題**
> $(2x - 3)^2$ を展開せよ。
>
> 解: $(2x)^2 - 2 \cdot 2x \cdot 3 + 3^2 = 4x^2 - 12x + 9$

> **💡 ポイント**
> 「**展開してから整理**」より「**整理してから展開**」が速い。複雑な式は **同類項をまとめてから** 公式を当てる。

### 1.2 因数分解

#### 1.2.1 因数分解とは

**展開の逆**。多項式を「**いくつかの式の積**」に直す操作。

例:
- 展開: $(x + 2)(x + 3) = x^2 + 5x + 6$
- 因数分解: $x^2 + 5x + 6 = (x + 2)(x + 3)$

これは方程式を解くとき、関数のグラフを描くとき、すべての場面で使う **超基本テクニック**。

#### 1.2.2 共通因数をくくる

最も基本の手筋。

$$ax + ay = a(x + y)$$

例: $6x^2 - 9x = 3x(2x - 3)$

#### 1.2.3 たすき掛け（$x^2$ の係数が 1）

$x^2 + (a + b)x + ab = (x + a)(x + b)$

**コツ**: 定数項 $ab$ を割り出し、和が $a + b$ になるペアを探す。

例: $x^2 + 5x + 6$ を因数分解
- $6 = 1 \cdot 6 = 2 \cdot 3$
- $a + b = 5$ となるのは $2 + 3$
- よって $(x + 2)(x + 3)$

#### 1.2.4 たすき掛け（係数が 1 以外）

$acx^2 + (ad + bc)x + bd = (ax + b)(cx + d)$

例: $2x^2 + 7x + 3$
```
    1 ────────→ 3 ───→ 6
                          ＼
                           +  → 7
                          ／
    2 ────────→ 1 ───→ 1
```
$(2x + 1)(x + 3)$ と分解できます。

#### 1.2.5 公式による因数分解

| 形 | 因数分解 |
|---|---|
| $a^2 - b^2$ | $(a + b)(a - b)$ |
| $a^2 + 2ab + b^2$ | $(a + b)^2$ |
| $a^2 - 2ab + b^2$ | $(a - b)^2$ |
| $a^3 + b^3$ | $(a + b)(a^2 - ab + b^2)$ |
| $a^3 - b^3$ | $(a - b)(a^2 + ab + b^2)$ |

#### 1.2.6 置き換え

複雑な式は **塊で置き換える** と見やすくなります。

例: $(x^2 + x)^2 - 8(x^2 + x) + 12$
- $t = x^2 + x$ と置くと、$t^2 - 8t + 12 = (t - 2)(t - 6)$
- 元に戻すと $(x^2 + x - 2)(x^2 + x - 6)$
- さらに $= (x - 1)(x + 2)(x - 2)(x + 3)$

### 1.3 実数

#### 1.3.1 数の分類

```
複素数 (数学 II)
  └── 実数 ℝ
        ├── 有理数 ℚ
        │     ├── 整数 ℤ
        │     │     ├── 自然数 ℕ (0, 1, 2, ...)
        │     │     ├── 0
        │     │     └── 負の整数
        │     └── 整数でない有理数 (1/2 など)
        └── 無理数 (√2, π, e など)
```

- **有理数**: 整数 / 整数 で表せる数
- **無理数**: そう表せない数

#### 1.3.2 有理数と無理数

$\sqrt 2$ が無理数であることは、CS 教科書第 1 章でも背理法で示しました。

「**循環小数 ⇔ 有理数**」というルールも便利。
- $0.\overline{3} = 0.333\ldots = 1/3$ → 有理数
- $\pi = 3.14159\ldots$（循環しない） → 無理数

#### 1.3.3 絶対値

$|a|$ は「**$a$ の数直線上での原点からの距離**」。

$$|a| = \begin{cases} a & (a \geq 0) \\ -a & (a < 0) \end{cases}$$

例: $|3| = 3$, $|-5| = 5$, $|0| = 0$。

性質:
- $|a| \geq 0$
- $|a|^2 = a^2$
- $|ab| = |a| \cdot |b|$
- $|a + b| \leq |a| + |b|$（三角不等式）

### 1.4 平方根の計算

#### 1.4.1 復習

$\sqrt a$ は「**2 乗すると $a$ になる正の数**」。

- $\sqrt 4 = 2$
- $\sqrt 9 = 3$
- $\sqrt 2 \approx 1.414$

中学で覚えるべき値:
- $\sqrt 2 \approx 1.41$
- $\sqrt 3 \approx 1.73$
- $\sqrt 5 \approx 2.24$
- $\sqrt 7 \approx 2.65$

#### 1.4.2 計算ルール

$$\sqrt a \cdot \sqrt b = \sqrt{ab}, \quad \frac{\sqrt a}{\sqrt b} = \sqrt{\frac{a}{b}}, \quad \sqrt{a^2 b} = a \sqrt b \;(a \geq 0)$$

例: $\sqrt{12} = \sqrt{4 \cdot 3} = 2\sqrt 3$

#### 1.4.3 分母の有理化

「**分母にルートが残らないように**」変形。

$$\frac{1}{\sqrt 2} = \frac{\sqrt 2}{2}$$

分母が和・差なら **共役** を掛ける:
$$\frac{1}{\sqrt 2 + 1} = \frac{\sqrt 2 - 1}{(\sqrt 2 + 1)(\sqrt 2 - 1)} = \frac{\sqrt 2 - 1}{2 - 1} = \sqrt 2 - 1$$

これは「**和と差の積 → 平方差**」を使う典型例。

#### 1.4.4 二重根号

$\sqrt{a + b + 2\sqrt{ab}} = \sqrt a + \sqrt b$ の形で外せます。

例: $\sqrt{5 + 2\sqrt 6}$
- $5 = 2 + 3$, $\sqrt 6 = \sqrt{2 \cdot 3}$
- $\sqrt{2 + 3 + 2\sqrt{2 \cdot 3}} = \sqrt 2 + \sqrt 3$

### 1.5 1 次不等式

#### 1.5.1 不等式の性質

$a > b$ のとき:
- 両辺に同じ数を足してよい
- 両辺に **正の数** を掛けても OK
- 両辺に **負の数** を掛けると向きが反転！

例: $-2x > 6$ の両辺を $-2$ で割ると、$x < -3$（向きが変わる）。

#### 1.5.2 一次不等式の解き方

例: $3(x - 1) \leq 2x + 5$
- 展開: $3x - 3 \leq 2x + 5$
- 移項: $3x - 2x \leq 5 + 3$
- $x \leq 8$

#### 1.5.3 連立不等式

例: $\begin{cases} 2x - 1 > 3 \\ x + 4 \leq 10 \end{cases}$
- 上: $x > 2$
- 下: $x \leq 6$
- 共通範囲: $2 < x \leq 6$

数直線で描くと分かりやすい:
```
─────●═══════○─────
     2       6
   含まず   含む
```

#### 1.5.4 絶対値を含む不等式

$|x| < a$（$a > 0$） ⇔ $-a < x < a$

$|x| > a$ ⇔ $x < -a$ または $x > a$

例: $|2x - 3| \leq 5$ → $-5 \leq 2x - 3 \leq 5$ → $-1 \leq x \leq 4$

### 1.6 集合と命題（論理の基礎）

#### 1.6.1 集合

ものの集まりを `{ }` で書きます。

例: $A = \{1, 2, 3, 4, 5\}$, $B = \{2, 4, 6\}$

- $\cup$ (和): どちらかに含まれる
- $\cap$ (積): 両方に含まれる
- $\bar A$ (補集合): 含まれない

$A \cup B = \{1, 2, 3, 4, 5, 6\}$, $A \cap B = \{2, 4\}$

#### 1.6.2 必要条件と十分条件

- 「$p \Rightarrow q$」: $p$ は $q$ の **十分条件**、$q$ は $p$ の **必要条件**
- 「$p \Leftrightarrow q$」: 必要十分条件

例: 「$x = 2$ ならば $x^2 = 4$」は真。$x = 2$ は $x^2 = 4$ の **十分条件**。
逆「$x^2 = 4$ ならば $x = 2$」は偽（$x = -2$ もある）。

「**$p$ なら $q$ だが、$q$ なら $p$ とは限らない**」のとき、$p$ は十分条件だが必要条件ではない。

> **🌱 試してみよう**
> 「$xy = 0$ ⇔ $x = 0$ または $y = 0$」は同値か?
> → ✓ 同値（必要十分）

#### 1.6.3 否定

「すべての $x$ について $P(x)$」の否定 → 「ある $x$ で $P(x)$ でない」

「ある $x$ で $P(x)$」の否定 → 「すべての $x$ について $P(x)$ でない」

```
"全員"の否定 → "ある 1 人が違う"
"誰かがいる"の否定 → "全員違う"
```

#### 1.6.4 対偶と背理法

- **対偶**: 「$p \Rightarrow q$」と「$\neg q \Rightarrow \neg p$」は同値
- **背理法**: 結論を否定して矛盾を導く

## 第 2 章 2 次関数

ここから「**関数 = $y$ を $x$ で表す式**」を本格的に扱います。共通テスト数学 I の最頻出単元。

### 2.1 関数の復習と用語

「**$x$ を決めると $y$ が決まる**」関係を関数といい、$y = f(x)$ と書きます。

- **定義域**: $x$ の取れる範囲
- **値域**: $y$ の取りうる範囲

例: $f(x) = x^2$, 定義域 $\mathbb{R}$, 値域 $\{y : y \geq 0\}$。

### 2.2 2 次関数のグラフ

#### 2.2.1 基本形

$y = ax^2$（$a \neq 0$） は **放物線**。

```
a > 0 のとき:     a < 0 のとき:
    ＼  ／              ＿
     ＼／              ／  ＼
                     ／     ＼
```

$|a|$ が大きいほど **開きが狭く**、$|a|$ が小さいほど **開きが広い**。

#### 2.2.2 平行移動

$y = a(x - p)^2 + q$ は、$y = ax^2$ を $x$ 方向に $p$、$y$ 方向に $q$ 平行移動したもの。

頂点は $(p, q)$。

```
y = (x - 3)^2 + 2:
頂点 (3, 2)、下に凸の放物線
```

> **💡 ポイント**
> $x$ 方向は **符号が逆**。$(x - 3)$ で **右** に 3 動く。

#### 2.2.3 一般形と平方完成

一般形 $y = ax^2 + bx + c$ を **基本形** $y = a(x - p)^2 + q$ に直す操作を **平方完成** といいます。

手順:
1. $x^2$ の係数 $a$ をくくる
2. $(\text{括弧内の }x \text{ 係数 } /2)^2$ を足して引く
3. 平方の形にまとめる

例: $y = 2x^2 - 8x + 5$
- $= 2(x^2 - 4x) + 5$
- $= 2(x^2 - 4x + 4 - 4) + 5$
- $= 2((x - 2)^2 - 4) + 5$
- $= 2(x - 2)^2 - 8 + 5$
- $= 2(x - 2)^2 - 3$

頂点は $(2, -3)$。

#### 2.2.4 グラフの描き方

1. 平方完成して頂点を求める
2. 開きの向き（$a$ の符号）
3. 軸（$x = p$）
4. $y$ 切片（$x = 0$ を代入）
5. $x$ 切片（$y = 0$ の解）

### 2.3 2 次方程式

#### 2.3.1 解の公式

$ax^2 + bx + c = 0$（$a \neq 0$）の解:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

**判別式** $D = b^2 - 4ac$:
- $D > 0$: 異なる 2 つの実数解
- $D = 0$: 重解
- $D < 0$: 実数解なし（複素数解、数学 II で扱う）

#### 2.3.2 解の公式の導出（理解のため）

$ax^2 + bx + c = 0$ を $a$ で割って平方完成:
$$x^2 + \frac{b}{a}x + \frac{c}{a} = 0$$
$$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$$
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

**「平方完成 → 解の公式」を 1 度自分の手で導いておく** と忘れません。

#### 2.3.3 解と係数の関係

$ax^2 + bx + c = 0$ の解を $\alpha, \beta$ とすると:
$$\alpha + \beta = -\frac{b}{a}, \quad \alpha \beta = \frac{c}{a}$$

これは「**解を直接求めずに**、解についての情報を引き出す」ときに便利。

### 2.4 2 次関数と 2 次不等式

#### 2.4.1 2 次関数の最大・最小

平方完成 $y = a(x - p)^2 + q$ から:
- $a > 0$: $x = p$ で **最小値** $q$
- $a < 0$: $x = p$ で **最大値** $q$

**定義域に制限がある場合**は、頂点が範囲内か外かで場合分け。

例: $y = x^2 - 4x + 3$ ($1 \leq x \leq 5$)
- 平方完成: $y = (x - 2)^2 - 1$、頂点 $(2, -1)$
- $x = 2$ は範囲内 → 最小値 $-1$
- 端点の値: $f(1) = 0$, $f(5) = 8$
- 最大値 $8$（$x = 5$）

#### 2.4.2 2 次不等式

$ax^2 + bx + c > 0$ のような不等式。グラフで考えるのが王道。

例: $x^2 - 5x + 6 > 0$
- 因数分解: $(x - 2)(x - 3) > 0$
- $y = (x-2)(x-3)$ のグラフは下に凸、$x = 2, 3$ で軸を切る
- $y > 0$ となる範囲は $x < 2$ または $x > 3$

```
   y
    \   ／
     \ ／
   ──┼──●──●──→ x
        2  3
```

#### 2.4.3 判別式と 2 次不等式

$ax^2 + bx + c > 0$ がすべての $x$ について成り立つ条件:
- $a > 0$ かつ $D < 0$

例: $x^2 + kx + 4 > 0$ が常に成り立つ $k$ の範囲
- $D = k^2 - 16 < 0$
- $-4 < k < 4$

> **💡 共通テスト頻出**
> 「すべての $x$ について成り立つ」「ある $x$ で成り立つ」のパターンは超頻出。グラフで「軸を切らない」「軸を切る」と判断。

## 第 3 章 図形と計量 — 三角比

中学のピタゴラスの定理を発展させた、図形を **数値で扱う** 武器。

### 3.1 三角比とは

直角三角形を考えます。

```
         |\
         | \
       b |  \ c (斜辺)
         |   \
         |  θ \
         +─────
            a
```

角 $\theta$ について 3 つの比を定義:

$$\sin\theta = \frac{\text{対辺}}{\text{斜辺}} = \frac{b}{c}$$
$$\cos\theta = \frac{\text{隣辺}}{\text{斜辺}} = \frac{a}{c}$$
$$\tan\theta = \frac{\text{対辺}}{\text{隣辺}} = \frac{b}{a}$$

#### 3.1.1 覚え方

「**サイン・コサイン・タンジェント**」を筆記体で書くと、$s$ は反対 / 斜辺、$c$ は隣 / 斜辺、$t$ は反対 / 隣を指す形になります。

### 3.2 三角比の値

#### 3.2.1 30°, 45°, 60° は暗記

| 角 | $\sin$ | $\cos$ | $\tan$ |
|---|---|---|---|
| 30° | $1/2$ | $\sqrt 3 / 2$ | $1/\sqrt 3$ |
| 45° | $\sqrt 2 / 2$ | $\sqrt 2 / 2$ | $1$ |
| 60° | $\sqrt 3 / 2$ | $1/2$ | $\sqrt 3$ |

#### 3.2.2 覚え方の図

```
30°-60°-90° 三角形:           45°-45°-90° 三角形:
        |\                              |\
        | \                             | \
      2 |  \ √3                       1 |  \ √2
        |   \                           |   \
        +───\                           +────
         1                                1
```

辺の比 $1:\sqrt 3:2$ と $1:1:\sqrt 2$ を覚えれば、三角比は復元できます。

### 3.3 三角比の拡張

90° 以上の角にも三角比を拡張します。**単位円** で考えます。

```
     y
     |
     ●(cosθ, sinθ)
    /|
   / |
  / θ|
 +───●───→ x
     (1, 0)
```

半径 1 の円上の点 $(x, y)$ について:
- $\cos\theta = x$
- $\sin\theta = y$
- $\tan\theta = y/x$

これで 0° 〜 180°（数学 I の範囲）、さらに 360° まで定義可能。

#### 3.3.1 主要な値

| $\theta$ | 0° | 90° | 120° | 135° | 150° | 180° |
|---|---|---|---|---|---|---|
| $\sin$ | 0 | 1 | $\sqrt 3/2$ | $\sqrt 2/2$ | 1/2 | 0 |
| $\cos$ | 1 | 0 | $-1/2$ | $-\sqrt 2/2$ | $-\sqrt 3/2$ | $-1$ |
| $\tan$ | 0 | × | $-\sqrt 3$ | $-1$ | $-1/\sqrt 3$ | 0 |

#### 3.3.2 重要な公式

- $\sin^2\theta + \cos^2\theta = 1$
- $\tan\theta = \sin\theta / \cos\theta$
- $1 + \tan^2\theta = 1/\cos^2\theta$

最初の式は「**ピタゴラスの定理**」そのもの。半径 1 の円上で $x^2 + y^2 = 1$。

#### 3.3.3 補角の公式

$\sin(180° - \theta) = \sin\theta$
$\cos(180° - \theta) = -\cos\theta$
$\tan(180° - \theta) = -\tan\theta$

### 3.4 正弦定理・余弦定理

#### 3.4.1 正弦定理

三角形 ABC で、向かい合う辺・角を $a, A$ などとすると:

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

$R$ は外接円の半径。

**使い所**: 「**辺・対角**」が出てくる問題。

#### 3.4.2 余弦定理

$$a^2 = b^2 + c^2 - 2bc\cos A$$

**使い所**: 「**2 辺と挟む角**」「**3 辺から角**」。

ピタゴラスの定理を一般化したもの（$A = 90°$ なら $\cos A = 0$ で $a^2 = b^2 + c^2$）。

#### 3.4.3 三角形の面積

$$S = \frac{1}{2} bc \sin A$$

「**2 辺 × 挟む角の sin / 2**」。

ヘロンの公式（参考）:
$$S = \sqrt{s(s-a)(s-b)(s-c)}, \quad s = \frac{a+b+c}{2}$$

### 3.5 図形への応用

- 円に内接する四角形（対角の和 180°、トレミーの定理）
- 三角形の内接円・外接円
- 空間図形（柱・錐・球の体積・表面積）

> **💡 共通テスト頻出**
> 「正弦定理と余弦定理の使い分け」が頻出。**「対角がペアで出るかどうか」** で判断するとよい。

## 第 4 章 データの分析

数学 I の最後の単元。共通テストでも 1 大問占めるので、軽視しないこと。

### 4.1 代表値

「**データ全体を 1 つの数値で表す**」もの。

| 名前 | 内容 |
|---|---|
| 平均値 | データの和 / 個数 |
| 中央値 | 並べて真ん中 |
| 最頻値 | 一番多く出る値 |

平均はすべての値に影響されるが、**外れ値に弱い**。中央値は外れ値に強い。

例: 5 人の年収 $\{200, 250, 300, 350, 10000\}$（単位万円）
- 平均: $2220$ 万円
- 中央値: $300$ 万円

「平均年収」と聞いたら、外れ値の影響を疑うクセを。

### 4.2 散らばりの指標

#### 4.2.1 分散と標準偏差

$$\text{分散} = \frac{1}{n}\sum_{i=1}^n (x_i - \bar x)^2$$

$$\text{標準偏差} = \sqrt{\text{分散}}$$

「**ばらつきの大きさ**」を測る。

例: $\{2, 4, 6\}$
- 平均: 4
- 分散: $\frac{(2-4)^2 + (4-4)^2 + (6-4)^2}{3} = 8/3$
- 標準偏差: $\sqrt{8/3} \approx 1.63$

#### 4.2.2 四分位範囲

データを 4 等分する点:
- 第 1 四分位数 ($Q_1$)
- 中央値 = 第 2 四分位数 ($Q_2$)
- 第 3 四分位数 ($Q_3$)

**四分位範囲** = $Q_3 - Q_1$（中央 50% の幅）。

#### 4.2.3 箱ひげ図

```
     ┌─────┐
─────┤     ├─────
     └─────┘
     Q1  Q3
最小値    中央値     最大値
```

**外れ値検出** の標準ツール。

#### 4.3.2 相関係数

$$r = \frac{\text{共分散}}{\sigma_x \sigma_y}$$

範囲: $-1 \leq r \leq 1$
- $r = 1$: 完全な正の相関
- $r = 0$: 無相関
- $r = -1$: 完全な負の相関

注意: **相関は因果ではない**。アイス売上と熱中症は相関するが、両方とも気温が原因。

### 4.4 仮説検定の考え方（新課程）

共通テストの新傾向。「**観測されたデータが偶然か、有意な差か**」を考える初歩。

例: コインを 100 回投げて表が 65 回出た → これは偶然?
- 帰無仮説: 公正なコイン（表 50%）
- 帰無仮説のもとでの確率を計算
- 確率が低ければ「**有意**」と判断

共通テストでは確率分布表が与えられ、与えられた基準（5% など）で判断する形式が多いです。
